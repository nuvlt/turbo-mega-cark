# 🔥 Turbo Mega Çark

Modern dark casino çark oyunu. React + Vite ile geliştirildi.

## 🎮 Oyun Özellikleri

- **16 dilimli çark**: %62.5 çarpan, %37.5 mini oyun
- **3 Mini Oyun**: Quick Match (kart eşleştirme), Guess Next (tahmin), Combo Match (slot)
- **RTP**: ~%75 hedefli, server-side outcome kontrolü
- **1.000 TL** başlangıç bakiyesi, 5–200 TL bet aralığı
- Canlı RTP takibi ve tur geçmişi

## 🚀 Kurulum

```bash
npm install
npm run dev
```

## 📦 Deploy (Vercel)

1. GitHub'a push edin
2. [vercel.com](https://vercel.com) → **New Project** → Repo'yu import edin
3. Vercel otomatik olarak Vite projesini tanır ve deploy eder

Veya Vercel CLI ile:
```bash
npm i -g vercel
vercel
```

## 🏗️ Build

```bash
npm run build   # dist/ klasörüne üretir
npm run preview # build önizleme
```
